## 1. What is Tex2Film?

### One-line Definition

> **Tex2Film is an AI cinematic studio that puts a complete film pre-production pipeline inside a web browser.**

### The 7-Stage Pipeline

```
📝 Narrative  →  🎬 Director's Brief  →  🎭 Elements  →  👁️ Vision
   Script           Visual DNA           Characters        Decoupage

→  🖼️ Storyboard  →  🎞️ Assembly  →  📦 Export
     Shot-by-Shot      NLE Timeline     Rendered MP4
```

Each stage of this pipeline represents a real part of cinematic pre-production that normally takes weeks and tens of thousands of dollars. Tex2Film reduces it to hours.

### Key Capabilities

| Capability | Description | Closest Competitor |
|------------|-------------|-------------------|
| **AI Script Generator** | LLM writes screenplay from a single idea | ChatGPT (no pipeline) |
| **Director's Brief** | Defines visual DNA: color, light, lens, reference film | **None** |
| **Auto Decoupage** | Converts screenplay to shot list with 180° rule, raccord, J-cut | **None** |
| **Character Consistency** | Maintains character faces across all shots (Kling Elements API) | Runway (limited) |
| **AI Image Generation** | Kling v3-Omni with continuity context | Midjourney, Kling |
| **Image-to-Video** | Converts any shot to video clip with camera movement | Runway, Pika, Kling |
| **TTS + SFX** | Dialogue and sound effects via ElevenLabs | ElevenLabs (standalone) |
| **NLE Timeline** | Browser-based editing with video sync | Adobe (standalone) |
| **FFmpeg Export** | Server-side rendering to MP4 with audio mix | **None** |
| **LangGraph Agent** | AI assistant with human-in-the-loop at every stage | **None** |

### Why "Director-First"?

All competitors use a "Prompt → Output" logic. This approach is unsustainable because:

1. The user has no control
2. Outputs are unrelated between shots
3. The final result is a slideshow, not a film

Tex2Film uses a **"Director → Cinematographer → Editor"** logic: the user is the director, the AI is the production assistant. This philosophical difference translates into a differentiated product that exists nowhere else.
